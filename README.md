# InsurMinds AutoQuote Chatbot

## Descrição

Este projeto tem como objetivo desenvolver um chatbot de atendimento para seguros automobilísticos, capaz de responder perguntas frequentes de segurados e realizar uma simulação de cotação de seguro auto.

O chatbot foi desenvolvido como parte da atividade obrigatória do curso InsurMinds, com foco na aplicação de inteligência artificial, automação de atendimento e organização de fluxos conversacionais no setor de seguros.

## Objetivos

- Automatizar o atendimento inicial ao segurado
- Responder dúvidas frequentes sobre seguro automobilístico
- Simular uma cotação acadêmica com base em fatores de risco
- Direcionar o usuário para atendimento humano quando necessário
- Documentar o processo de desenvolvimento, testes e execução

## Funcionalidades

- Atendimento conversacional
- Base de perguntas frequentes sobre seguro auto
- Simulação de cotação acadêmica
- Explicação de conceitos como franquia, cobertura, sinistro e assistência
- Registro de evidências de execução

## Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- CSV como base de dados
- LLM para geração de respostas
- Mermaid para diagramas
